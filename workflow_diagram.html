Cervical Cytology Classification Workflow

Input Image
Cervical cytology image (Pap smear or LBC)
Inception v3
Pre-trained CNN for feature extraction
Xception
Uses depthwise separable convolutions
DenseNet-169
Dense connectivity for rich features
Generate probability scores for each class
Fuzzy Rank Computation
Apply tanh and exp functions to compute ranks
R_k^i1 = 1 - tanh((P_k^i - 1)²/2), R_k^i2 = 1 - exp(-(P_k^i - 1)²/2)
Rank Score
Multiply ranks: RS_k^i = R_k^i1 × R_k^i2
Fused Score
Sum ranks across classifiers: FS_k = Σ RS_k^i
Final Prediction
Select class with minimum fused score
Class = min(FS_k)
